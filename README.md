# Currículo Escritora

Landing page para currículo de escritora, com navegação por scroll spy e seção de portfólio com modal e carrossel para detalhes.

🔗 [portfolio.madueoslivros.com.br](https://portfolio.madueoslivros.com.br)

## Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- shadcn/ui (Radix UI)

## Destaques técnicos

- **Scroll spy**: menu de navegação atualiza a seção ativa conforme o scroll da página
- **Seção portfólio** com modal e carrossel para melhor visualização de cada item
- **Componentes reutilizáveis** para padronizar layout e estilo
- **Responsivo**, com versão mobile simplificada
- **Deploy automatizado** via GitHub Actions para Hostinger

## Rodando localmente

```bash
npm install
npm run dev
```
